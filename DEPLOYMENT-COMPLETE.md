## 🔄 Workflow (6 Phases)

```
User Query: "fleischrezepte"
    ↓
[PHASE 1] RAG-Daemon Search
    └─ Find relevant PDFs via FTS5 + Kiwix
    └─ Quick answer from 14b
    ↓
[PHASE 2] Content Extraction
    └─ Run extract-content.py on top 5 PDFs
    └─ Auto-detect content types
    └─ Collect structured data
    ↓
[PHASE 3] Content Synthesis
    └─ Run synthesize-content.py
    └─ Type-specific 14b prompts
    └─ Create complete, actionable response
    ↓
[PHASE 4] Report Generation
    └─ Create Markdown report
    └─ Include metadata + sources
    ↓
[PHASE 5] HTML Conversion
    └─ Beautiful, printable HTML
    └─ Responsive CSS styling
    └─ Formatted for PDF export
    ↓
[PHASE 6] Browser Display
    └─ Auto-open in DISPLAY=:1
    └─ Ready to read/print/archive
    ↓
Output: Actionable, complete content
```
